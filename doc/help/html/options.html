Options

Titan Application Version

Set the Titan application binary files to use when launching a Titan model run. The setting allows the user flexibility to use Titan application files that may have been modified by the user for testing purposes.

Use External Titan Binaries

When set to False, the Titan application binary files specified by the user's PATH environment variable will be used when a Titan run is launched from this interface.
When set to True, the Titan application binary files at the location specified by the titan binary directory entered on this panel will be used when a Titan run is launched from this interface.

Titan Binary Directory

The fully qualified path where the Titan application binary files are located. The field is not required if the use of external binaries is disabled. The value can be set by using the directory chooser, which is displayed by pushing the "Titan Binary Directory" button, or entering the directory in the text field.
